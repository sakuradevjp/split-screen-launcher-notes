Split Screen Launcher

한 번의 탭으로 두 앱을 동시에. 작업이 더 빠르고 매끄럽게.

Google Play에서 받기 개발자 노트
★ 사용자 리뷰

데모 영상

  • 최신 Android 버전에 지속적으로 대응합니다.
  • 광고 없음

Split Screen Launcher는 자주 사용하는 앱 조합을 페어로 저장하고, 평소의 두 앱 화면을 한 번의 탭으로 즉시 불러올 수 있는 런처 앱입니다.
매번 앱을 골라 분할 화면을 설정할 필요는 더 이상 없습니다. 페어를 만들고, 탭하기만 하면 됩니다.

자주 쓰는 사용 예

영상 + SNS: YouTube를 보면서 X 확인하기
검색 + 메모: Chrome으로 검색하며 메모하기
업무 + 연락: Gmail과 Slack을 나란히 띄워 동시에 대응하기
비교 + 계산: 브라우저와 계산기를 나란히 배치하기
이동 + 연락: 지도를 보면서 채팅으로 위치 공유하기

주요 기능

  • 앱 페어의 생성·저장·홈 화면 바로가기 만들기·원탭 실행
  • 15개 언어 지원, 앱 내 언어 전환 가능 EN, JA, KO, ZH-CN, ZH-TW, ES, PT-BR, FR, DE, IT, RU, AR, HI, TH, VI
  • 다크 모드 지원
  • 백업/복원
  • 가볍고 단순한 설계

개발자 응원

무료 버전에서는 최대 2개의 페어를 저장할 수 있습니다. Pro로 업그레이드하면 무제한 페어, 홈 화면 바로가기, 폴더 정리, 색상 커스터마이즈가 잠금 해제됩니다. 광고 없는 앱 제공과 1인 개발의 지속을 응원해 주세요!

접근성(Accessibility) API에 관하여

Android 9~12에서는 분할 화면을 실행하기 위해 접근성 서비스를 사용합니다. 이 서비스는 분할 화면 실행에만 사용되며, 화면의 내용을 수집하거나 전송하지 않습니다. Android 12L 이상에서는 접근성 서비스가 필요하지 않습니다.
