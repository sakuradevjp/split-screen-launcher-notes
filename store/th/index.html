Split Screen Launcher

เปิดสองแอปพร้อมกันด้วยแตะครั้งเดียว ทำงานได้ลื่นไหลขึ้น

ดาวน์โหลดบน Google Play บันทึกของนักพัฒนา
  • รองรับ Android เวอร์ชันล่าสุดอย่างต่อเนื่อง
  • ไม่มีโฆษณา

Split Screen Launcher ให้คุณบันทึกคู่แอปที่ใช้ประจำ และเรียกใช้งานหน้าจอสองแอปแบบเดิมได้ทันที
ไม่ต้องตั้งค่าซ้ำ แค่สร้างคู่แล้วแตะ

การใช้งานทั่วไป

วิดีโอ + โซเชียล: ดู YouTube ขณะเลื่อนดู X
ค้นคว้า + โน้ต: ท่องเว็บใน Chrome ขณะจดบันทึก
ทำงาน + สื่อสาร: วาง Gmail และ Slack เคียงข้างกัน
เปรียบเทียบ + คำนวณ: วางเบราว์เซอร์ข้างเครื่องคิดเลข
นำทาง + แชท: แชร์ตำแหน่งขณะดูแผนที่

คุณสมบัติหลัก

  • สร้าง บันทึก และเปิดคู่แอปด้วยแตะครั้งเดียว
  • รองรับ 15 ภาษา พร้อมเปลี่ยนภาษาในแอป EN, JA, KO, ZH-CN, ZH-TW, ES, PT-BR, FR, DE, IT, RU, AR, HI, TH, VI
  • รองรับโหมดมืด
  • สำรองและกู้คืนข้อมูล
  • ออกแบบเบาและเรียบง่าย

เกี่ยวกับ Accessibility API

บน Android 9–12 แอปนี้ใช้บริการการเข้าถึงเพื่อเปิดโหมดแบ่งหน้าจอ บริการนี้ใช้เฉพาะสำหรับการเปิดหน้าจอแบ่งเท่านั้น และไม่เก็บรวบรวมหรือส่งเนื้อหาใด ๆ บนหน้าจอ บน Android 12L ขึ้นไป ไม่จำเป็นต้องใช้บริการการเข้าถึง
