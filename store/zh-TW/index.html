Split Screen Launcher 可儲存您常用的應用程式組合，並立即啟動熟悉的雙應用分割畫面配置。
無需重複設定，建立一次即可。

常見用途

影片 + 社群：邊看 YouTube 邊滑 X
查閱 + 筆記：邊用 Chrome 瀏覽邊記錄
工作 + 溝通：Gmail 和 Slack 並排使用
對比 + 計算：瀏覽器和計算機並排放置
導航 + 聊天：邊看地圖邊分享位置

主要功能

  • 一鍵建立、儲存、釘選至主畫面和啟動應用程式組合
  • 支援 15 種語言，可在應用程式內切換 EN, JA, KO, ZH-CN, ZH-TW, ES, PT-BR, FR, DE, IT, RU, AR, HI, TH, VI
  • 深色模式
  • 備份與還原
  • 輕量簡潔的設計

支持開發者

免費版最多可儲存 2 個組合。升級至 Pro 即可解鎖無限組合、主畫面捷徑、資料夾整理和自訂顏色。讓應用保持無廣告，支持獨立開發！

關於 Accessibility API

在 Android 9–12 上，本應用程式使用無障礙服務來啟動分割畫面模式。該服務僅用於啟動分割畫面，不會收集或傳輸任何螢幕內容。在 Android 12L 及以上版本中，不需要無障礙服務。

免費,無廣告。在 Google Play 試試看。
