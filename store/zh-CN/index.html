Split Screen Launcher 可保存你常用的应用组合，并立即打开熟悉的双应用分屏布局。
无需重复设置，创建一次即可。

常见用途

视频 + 社交：边看 YouTube 边刷 X
查阅 + 笔记：边用 Chrome 浏览边记录
工作 + 沟通：Gmail 和 Slack 并排使用
对比 + 计算：浏览器和计算器并排放置
导航 + 聊天：边看地图边分享位置

主要功能

  • 一键创建、保存、固定到主屏幕和启动应用组合
  • 支持 15 种语言，可在应用内切换 EN, JA, KO, ZH-CN, ZH-TW, ES, PT-BR, FR, DE, IT, RU, AR, HI, TH, VI
  • 深色模式
  • 备份与恢复
  • 轻量简洁的设计

支持开发者

免费版最多可保存 2 个组合。升级到 Pro 即可解锁无限组合、主屏幕快捷方式、文件夹整理和自定义颜色。让应用保持无广告，支持独立开发！

关于 Accessibility API

在 Android 9–12 上，本应用使用无障碍服务来启动分屏模式。该服务仅用于启动分屏，不会收集或传输任何屏幕内容。在 Android 12L 及以上版本中，不需要无障碍服务。
