Uso de la Accessibility API

En Android 9–12, esta app utiliza un Servicio de Accesibilidad para activar el modo de pantalla dividida. El servicio se utiliza únicamente para iniciar la pantalla dividida y no recopila ni transmite ningún contenido de la pantalla. En Android 12L y versiones posteriores, el Servicio de Accesibilidad no es necesario.
