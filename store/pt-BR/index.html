Pesquisa + Notas: Navegar no Chrome enquanto faz anotações

Navegação + Chat: Compartilhar localização enquanto consulta o mapa

Apoie o Desenvolvedor

A versão gratuita permite até 2 pares. Atualize para o Pro para pares ilimitados, atalhos na tela inicial, organização em pastas e cores personalizadas. Mantenha o app sem anúncios e apoie o desenvolvimento independente!

Uso da Accessibility API

No Android 9–12, este app utiliza um Serviço de Acessibilidade para ativar o modo de tela dividida. O serviço é usado exclusivamente para iniciar a tela dividida e não coleta nem transmite nenhum conteúdo da tela. No Android 12L e versões posteriores, o Serviço de Acessibilidade não é necessário.
