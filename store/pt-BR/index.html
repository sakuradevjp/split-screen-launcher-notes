Uso da Accessibility API

No Android 9–12, este app utiliza um Serviço de Acessibilidade para ativar o modo de tela dividida. O serviço é usado exclusivamente para iniciar a tela dividida e não coleta nem transmite nenhum conteúdo da tela. No Android 12L e versões posteriores, o Serviço de Acessibilidade não é necessário.
