Split Screen Launcher

2つのアプリを、ワンタップで同時に。いつもの作業が、もっとスムーズに。

Split Screen Launcher は、よく使うアプリの組み合わせをペアとして保存し、
タップひとつで「いつもの2画面」をすぐに呼び出せるランチャーアプリです。
毎回アプリを選んで分割表示を設定する手間は、もう要りません。
ペアを作って、タップするだけ。

よくある使い方

動画＋SNS：YouTube を観ながら X をチェック
調べもの＋メモ：Chrome で検索しながらノートに記録
仕事＋連絡：Gmail と Slack を並べて同時に対応
比較＋計算：ブラウザと電卓を横に並べる
移動＋連絡：マップを見ながらチャットで共有

主な機能

  • アプリペアの作成・保存・デスクトップショートカット作成・ワンタップ起動
  • 15言語対応・アプリ内言語切替 EN, JA, KO, ZH-CN, ZH-TW, ES, PT-BR, FR, DE, IT, RU, AR, HI, TH, VI
  • ダークモード対応
  • バックアップ／復元
  • 軽量でシンプルな設計

開発者を応援

無料版ではペアを2個まで保存できます。Proにアップグレードすると、ペア無制限・デスクトップショートカット・フォルダ整理・カラーカスタマイズが解放されます。広告なしでの提供と個人開発の継続を応援してください！

ユーザー補助（Accessibility）APIについて

Android 9〜12 では、分割画面を起動するためにユーザー補助サービスを使用します。このサービスは分割画面の起動のみに使用され、画面の内容を収集・送信することはありません。Android 12L 以降ではユーザー補助サービスは不要です。
