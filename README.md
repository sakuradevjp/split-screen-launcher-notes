## What is Split Screen Launcher?

An Android app pair launcher that **still works on Android 15 and 16** — when most other split-screen utilities have stopped functioning. No ads, no Accessibility permission required, only 2.8 MB.

- 🪟 Save app pairs and launch both with one tap
- ✅ Compatible with Android 12L and up, including 15/16
- 🪶 Tiny: 2.8 MB
- 🔒 No Accessibility service, no permissions abuse
- 🆓 Free, no ads

**Get it on Google Play**: https://play.google.com/store/apps/details?id=com.ryodev.splitscreen

**Read the docs (live page)**: https://sakuradevjp.github.io/split-screen-launcher-notes/
